hexo 指南

Posted on Dec 9, 2021

注意：hexo 过于麻烦，相关替代品 hugo、wordpress。

hexo 主体安装

1npm install -g hexo
2# 把hexo加入path
3hexo init blog       # 你的网站在本地的目录
4cd blog
5npm install
6hexo server          # 服务器部署 或者 本地看效果

Github pages 配置

  1. 流程很简单，首先安装hexo-deployer-git插件
1# blog 目录下
2npm install hexo-deployer-git --save
  1. 修改根目录_config.yml
 1deploy:
 2  type: git
 3  repo: https://github.com/ao-blog/ao-blog.github.io
 4  # example, https://github.com/hexojs/hexojs.github.io
 5  branch: master
 6 
 7# URL
 8## Set your site url here. For example, if you use GitHub Page, set url as 'https://username.github.io/project'
 9url: https://ao-blog.github.io
10# 这一步注意是 https://username.github.io/project 地址
11
12## 图片
13post_asset_folder: true # 是否生成同名文档
  1. 选个酷炫主题（或者自己写/改）
1git clone https://github.com/probberechts/hexo-theme-cactus.git themes/cactus
2# 目前cactus，修改对应theme
  1. 编辑文章
1hexo new [layout] <name> # 不写就默认使用日期规则
2hexo draft <name> # 打个草稿，实际使用Github Pages的基本用不到
3hexo publish [layout] <name> # 就是简单移动一下
4# layout 可以省略
5
6# 注意 如果直接修改文件

layout 种类

index 首页 分区
post 文章 index
page 分页 index
archive 归档 index
category 分类归档 archive
tag 标签归档 archive
