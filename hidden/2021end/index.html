一点都不蓝色的2021

Posted on Dec 15, 2021

今年对于我而言，已经结束了。

虽然25号还有全国研究生统一招生考试，这一刑场等着我去受刑。（龙宫院最后的战斗，最终决战的魔女之夜）

但是学院里，实际上已经放了假，给考研学生的慰问品刚刚发放到位，并通知最后3周的项目教学线上进行，同时只提交电子报告。换而言之，只要胆子大，现在就能买张票，明天就能看到家里的太阳了。

然后，明年这个点再战？我深有预感，假设今年做了逃兵，那么明年此刻，我定然也是逃兵。所以，逃げちゃ駄目だ!逃げちゃ駄目だ!!逃げちゃ駄目だ!!!

即使知道必败无疑，还是只能上了。残念的是，这里就是我的战场。

我躺在家里的床上，仔细思索这四年，到底学了什么——就结论而言——什么都没学到。

 1随便举例 (;´д｀）
 2数据结构、算法 -> 手写反转红黑树的故事永不过时
 3操作系统 -> 详细解释一下，系统从加电后开始的启动过程，汇编不行，源码看不懂啊
 4深度学习 -> 调包侠，调参侠 -> import tensorflow as torch
 5Web C# QT -> 做的都是玩具，中专都比我行。简而言之，新世纪API战士
 6Android -> 是 计算器app 还是 天气app ORZ
 7linux -> 不是Arch爱好者，Windows最香了。BTW I use Windows.
 8Java -> 本人最讨厌语言，没有之一
 9计算机网络 -> 详解TCP/IP？不会啊。Unix网络编程？不会，告辞
10设计模式 -> 会飞的橡皮鸭
11数据库 -> 这辈子不碰Oracle
12汇编 -> 8086都够呛，更别提x86 & x64了
13C C++ -> 指针函数，函数指针，虚拟继承，STL背不了，告辞
14计算机组成原理 -> 数电没学好，原理搞不了，科普水平
15编译原理 -> 考完就忘，什么LR(0)、SLR、LR？
16图形学、密码学、网安、计算机视觉 -> 学了个寂寞、有的课都没开、自学
17高数、数分、线代 -> 写题不能

这四年基本什么都没学到，四年成为了网络中的游魂。/home/ao就是家目录，/usr/bin是手脚，chrome是眼睛。动画则是粮食。

那么接下来，该做什么呢？面对研究生考试大魔王？虽然下学期有毕设开题，但是我实际进展已经快做完了，近似于结题答辩。最后一个学期估计也是颓废的，我肯定没有勇气再来面对我失败的考试了。一鼓作气、再而衰、三而竭。等待我的必然是无法完成的目标和比今年更甚的痛苦。

老师说：“期待你来XX！不会只有那一次，一定要向下一次高光努力。”学院群里说：“考研倒计时，乾坤未定，你我都是黑马。”（按照去年的数据，3/36的黑马是吧）而我，已知自己筋疲力竭，对于几乎没准备的我，命数已定。言语无法改变现状，但对于我足矣。

而年末恰好有两部作品，探讨了努力，分别是国王排名与蓝色时期，国王排名对我而言，优秀在于动人的刻画了弱者与不完美的爱，而后期作品节奏的崩坏，乃至给主角强行叠buff，走火影忍者甚至博人传的老路，是对前期努力，感情的最好打脸。烂尾在所难免。

而蓝色时期，却完全相反。相比于摆烂的我，主角凭借着天赋和努力，强大的毅力，恰好的环境，高超的应试技术，成功合格。实在是典范中的典范，王道作中的王道，堪称半部食梦者。看到最后，无地自容，颇为羡慕，剧中桑名真纪的落榜，意外却又无比真实。如果是高中，我可能还能再来，可是现在却变成了立刻躺平。

此篇总结也属于是“远野贵树式”虚空输出罢了。如同只有神知道的世界一般，我也可以说：我已经看到结局了。无论如何，摆在我面前的就一条路，要么立刻空血无伤击杀BOSS，要么复活再来，无尽循环。逃げちゃ駄目だ! EVA都完结了。我逃无可逃。动画TV版里可没有真希波这个角色。

无论如何，这里我是一秒也不想呆了。

但是去了XX就会好吗？估计和现在大同小异。依旧是这样，只是最后的战斗结束了罢了。
